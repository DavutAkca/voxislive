## Description
Voxis は Windows 向けのリアルタイム音声翻訳アプリです。PC で再生されている音 — 動画、ライブ配信、ゲーム、オンライン会議の相手の声 — を聞き取り、まだ話している最中に、あなたの言語で音声翻訳を再生します。字幕だけではありません。実際に、ライブで聞こえます。

人間の同時通訳者のように、Voxis は途切れずに翻訳し、数秒遅れを保ちながら、速さと正確さを自動で両立させます。

できること
• あらゆる動画・配信・映画を、ライブ音声であなたの言語に翻訳
• ゲーム、ボイスチャット、配信者をリアルタイムで理解
• オンライン会議や通話に対応 — 相手の声、さらには自分の声も双方向で翻訳
• ライブ字幕を読み、検索できる文字起こし履歴を保存（SRT / VTT / TXT に書き出し）

Voxis を選ぶ理由
• 70 以上の言語に対応
• ドライバー不要のシステム音声キャプチャ — 一方向翻訳に追加インストールは不要
• スマートダッキングが、翻訳が話す間だけ元の音声を下げます
• 誠実な低遅延設計 — 偽りの「瞬時」ではなく、通訳者のような自然な遅延
• プライバシー重視: 音声は翻訳のためだけに送信され、当社サーバーには保存されません
• 自分の鍵を使う（BYOK）テレメトリーゼロのオープンソース版を GitHub で提供

AI 搭載
翻訳と音声出力は、Google の Gemini ライブ翻訳 AI によって生成され、自然で同時的な音声から音声への結果を実現します。

はじめに
Voxis をインストールし、サインインして、言語を選び、開始を押すだけ — 新規アカウントには無料の翻訳分数が含まれます。いつでもアップグレード可能です。

Voxis は PC で音を再生するあらゆるアプリで動作します。プライバシーと規約: https://voxislive.com/privacy · https://voxislive.com/terms
