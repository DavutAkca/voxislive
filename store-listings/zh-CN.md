## Short description
Voxis 实时翻译你电脑的声音——视频、游戏和在线会议——并用你自己的语言以语音方式实时播放。支持 70 多种语言，由 AI 驱动，隐私优先。

## Description
Voxis 是一款适用于 Windows 的实时语音翻译应用。它会聆听你电脑上正在播放的声音——一段视频、一场直播、一款游戏，或在线会议中对方的声音——并在对方还在说话时，用你的语言播放语音翻译。不只是字幕：你真的能实时听到。

就像人类同声传译一样，Voxis 持续翻译，并保持几秒的延迟，自行在速度与准确度之间取得平衡。

你可以做什么
• 用实时语音把任何视频、直播或电影翻译成你的语言
• 实时理解游戏、语音聊天和主播
• 跟上在线会议和通话——双向翻译对方的声音，甚至你自己的声音
• 阅读实时字幕并保存可搜索的转写记录（导出为 SRT / VTT / TXT）

为什么选择 Voxis
• 支持 70 多种语言
• 无驱动的系统音频捕获——单向翻译无需额外安装
• 智能闪避在翻译说话时降低原始音量，不会被盖过
• 诚实的低延迟设计——没有虚假的"即时"，只有自然的传译延迟
• 隐私优先：你的音频仅用于翻译而传输，不会存储在我们的服务器上
• GitHub 上提供自带密钥（BYOK）、零遥测的开源版本

由 AI 驱动
翻译和语音输出由 Google 的 Gemini 实时翻译 AI 生成，带来自然、同步的语音到语音效果。

开始使用
安装 Voxis，登录，选择你的语言，然后点击开始——新账户包含免费翻译分钟数。随时可升级。

Voxis 适用于任何在你电脑上播放声音的应用。隐私与条款：https://voxislive.com/privacy · https://voxislive.com/terms
