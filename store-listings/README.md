## Field character limits (Microsoft Store)
- **Short title:** 50 · **Voice title:** 255
- **Short description:** 1,000
- **Description:** 10,000
- **Product features:** up to 20, ≤200 chars each
- **Search terms:** up to 7, ≤30 chars each (hidden; used for search ranking)
- **What's new in this version:** 1,500
- **Copyright and trademark info:** 200 · **Developed by:** 255

## Compliance reminders (baked into every file)
- AI disclosure present (Policy 11.16): translation "powered by Google Gemini".
- No third-party platform names (Netflix/Spotify/etc.) — generic "any video, stream, game".
- "Voxis" title carries no marketing/keywords (Policy 10.1.1).
- Honest latency framing ("a few seconds behind, like an interpreter").
