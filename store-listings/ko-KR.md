## Short description
Voxis는 PC의 오디오를 실시간으로 번역합니다 — 동영상, 게임, 온라인 회의 — 그리고 내 언어로 음성으로, 실시간으로 재생합니다. 70개 이상 언어, AI 기반, 개인정보 보호 우선.

## Description
Voxis는 Windows용 실시간 음성 번역기입니다. PC에서 재생되는 소리 — 동영상, 라이브 스트림, 게임, 또는 온라인 회의 상대방의 목소리 — 를 듣고, 아직 말하는 중에 내 언어로 음성 번역을 재생합니다. 단순한 자막이 아니라 실제로, 실시간으로 들립니다.

사람 동시통역사처럼 Voxis는 끊김 없이 번역하며 몇 초 뒤를 유지하면서 속도와 정확성을 스스로 조율합니다.

할 수 있는 것
• 모든 동영상, 스트림, 영화를 실시간 음성으로 내 언어로 번역
• 게임, 음성 채팅, 스트리머를 실시간으로 이해
• 온라인 회의와 통화를 따라가기 — 상대방, 나아가 내 목소리까지 양방향 번역
• 실시간 자막을 읽고 검색 가능한 대화 기록 보관 (SRT / VTT / TXT로 내보내기)

Voxis를 선택하는 이유
• 70개 이상 언어 지원
• 드라이버 없는 시스템 오디오 캡처 — 단방향 번역에는 추가 설치 불필요
• 스마트 더킹이 번역이 말하는 동안 원본 소리를 낮춰 줍니다
• 정직한 저지연 설계 — 가짜 "즉시"가 아니라 통역사 같은 자연스러운 지연
• 개인정보 보호 우선: 오디오는 번역을 위해서만 전송되며 당사 서버에 저장되지 않습니다
• 자체 키 사용(BYOK), 텔레메트리 제로의 오픈소스 버전을 GitHub에서 제공

AI 기반
번역과 음성 출력은 Google의 Gemini 실시간 번역 AI로 생성되어 자연스럽고 동시적인 음성-대-음성 결과를 제공합니다.

시작하기
Voxis를 설치하고 로그인한 뒤 언어를 선택하고 시작을 누르세요 — 신규 계정에는 무료 번역 시간이 포함됩니다. 언제든 업그레이드할 수 있습니다.

Voxis는 PC에서 소리를 재생하는 모든 앱과 함께 작동합니다. 개인정보 및 약관: https://voxislive.com/privacy · https://voxislive.com/terms

## Product features
- 모든 동영상, 스트림, 게임, 온라인 회의를 위한 실시간 음성 번역
- 번역을 내 언어로 음성으로, 실시간으로 듣기 — 단순 자막이 아님
- 70개 이상 언어 지원
- 양방향 회의 모드: 상대방과 내 목소리를 번역
- 검색 가능한 대화 기록과 SRT/VTT/TXT 내보내기를 갖춘 실시간 자막
- 자연스럽고 동시적인 번역을 위한 Google Gemini AI 기반
- 스마트 더킹이 번역 중 원본 소리를 낮춤
- 단방향 번역을 위한 드라이버 없는 시스템 오디오 캡처
- 개인정보 보호 우선: 오디오는 번역을 위해서만 전송, 저장 안 함
- 자체 키 사용(BYOK), 텔레메트리 제로의 오픈소스 버전
- PC에서 소리를 재생하는 모든 앱과 작동
